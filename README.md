### Akita 

## About 

Akita, like its namesake, is an early warning system for phishing and brand-impersonation. It is a Certificate Transparency monitor that watches public CT logs for newly issued certificates matching your domains or typosquat patterns.

This is version 0.0.0. And thus the README is intentionally sparse. I will update this as I build it out. The roadmap is up-to-date as of June 17, 2026.
